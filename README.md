# Aditya Pratama Putra - Portfolio

Website portfolio pribadi yang dibangun menggunakan framework Laravel.

## Fitur
- Beranda (Home)
- Tentang Saya (About)
- Resume (Pendidikan & Pengalaman)
- Proyek (Portfolio)
- Form Kontak

## Teknologi yang Digunakan
- Laravel
- Blade Templating
- Vanilla CSS
- JavaScript

## Cara Menjalankan
1. Clone repository ini.
2. Jalankan `composer install`.
3. Salin `.env.example` ke `.env` dan atur konfigurasi database (jika diperlukan).
4. Jalankan `php artisan key:generate`.
5. Jalankan `php artisan serve`.
6. Buka `http://127.0.0.1:8000` di browser.
